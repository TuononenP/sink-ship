Short description:
The main idea of the game is to try to sink opponent's ships. The player can only see a matrix that represents a sea and can't see the opponents ships. The idea is to guess where the opponent's ship could be located and try and hit that matrix cell. The end come is a hit or miss. The winner is the first one to sink opponent's every ship.

User requirements

Java JRE (Java Runtime Environment) 5 or newer must be installed into your system. JRE can be downloaded from www.java.com/getjava/.

To start the program just click Chess.jar file.
